# SECTION 3: Listing Checklist

## Pre-Publish Checklist

### Photos
- [ ] 5+ high-quality photos
- [ ] Main photo on white background
- [ ] Lifestyle/context photos
- [ ] Detail shots
- [ ] Scale reference (if applicable)
- [ ] Bright, natural lighting
- [ ] No watermarks

### Title
- [ ] Primary keyword in first 140 characters
- [ ] Relevant secondary keywords
- [ ] Clear product description
- [ ] No keyword stuffing
- [ ] Readable, not just keywords

### Description
- [ ] First 155 characters hook the reader
- [ ] Clear product details (size, material, etc)
- [ ] Keywords naturally incorporated
- [ ] Emotional benefits highlighted
- [ ] Care instructions included
- [ ] Shop policies referenced
- [ ] Call to action at end

### Tags
- [ ] 13 tags used
- [ ] Mix of broad and specific
- [ ] All relevant to product
- [ ] No misspellings

### Pricing
- [ ] Calculated cost + profit
- [ ] Competitor comparison done
- [ ] Etsy fees accounted for
- [ ] Shipping considered

### Variations
- [ ] All options clearly shown
- [ ] Photos for each variation
- [ ] Price differences noted

# SECTION 4: Title Templates

## 20+ Title Formulas

### 1. Classic Descriptive
```
[Material] + [Product Type] + [Key Feature] + [Use] + [Style]
Example: Handmade Leather Journal - Refillable - Travel Size - Rustic Design
```

### 2. Keyword Rich
```
[Primary Keyword] + [Secondary Keyword] + [Feature] + [Benefit]
Example: Custom Dog Collar - Personalized - Embroidery - Gift for Dog Lover
```

### 3. Gift Focused
```
[Gift For] + [Occasion] + [Product] + [Personalization]
Example: Anniversary Gift for Wife - Custom Photo Frame - 10x8 - Silver
```

### 4. Style Driven
```
[Vintage/Modern/Rustic] + [Product] + [Material] + [Room/Use]
Example: Modern Kitchen Wall Art - Metal - Farmhouse Style - Dining Room Decor
```

### 5. Problem Solver
```
[Solution] + For + [Problem] + [Product Type]
Example: Solution for Cluttered Desk - Wooden Desk Organizer - Bamboo
```

### 6. Bundle Title
```
[Product 1] + [Product 2] + [Product 3] + Bundle
Example: Candle Set - Wax Melts - Room Spray - Lavender Bundle - Spa Gift
```

### 7. Personalized
```
Custom [Product] - [Personalization Option] - [Details]
Example: Custom Name Sign - Up to 10 Letters - Nursery Decor - Wooden
```

### 8. Size Focused
```
[Size] + [Product] + [Material] + [Feature]
Example: Large Canvas Wall Art - 24x36 - Ready to Hang - Minimalist
```

### 9. Handmade Quality
```
Handmade [Product] - [Material] - [Feature] - [Use]
Example: Handmade Silver Earrings - Hypoallergenic - Dangle - Wedding
```

### 10. Seasonal
```
[Holiday] + [Product] + [Theme] + [Gift]
Example: Christmas Wreath - Holiday Door Decor - Red Ribbon - Gift Box
```

---

## Title Mistakes to Avoid

❌ Keyword stuffing
❌ ALL CAPS
❌ Too long (cut off in search)
❌ Irrelevant keywords
❌ Special characters only (@#$%)
❌ Missing key information

# SECTION 5: Description Templates

## Description Structure

### Opening Hook (First 155 characters)
```
[Emotional hook] + [Primary benefit] + [Product type]

Example: ✨ Transform your workspace with this handcrafted wooden desk organizer - the 
perfect solution for keeping your supplies organized and accessible!
```

---

### Section 1: Introduction
```
[Who is this for] + [What it is] + [Why it's special]

Tell your customers WHO this product is for and WHAT makes it special.
```

---

### Section 2: Product Details
```
📏 SIZE:
[Exact dimensions]

🎨 MATERIALS:
[Materials used]

👐 HANDCRAFTED:
[How it's made]

✨ FEATURES:
[Key features - 3-5 bullet points]
```

---

### Section 3: How to Use
```
📝 HOW TO ORDER:
[Step-by-step ordering instructions if custom]

🎁 PERFECT FOR:
[Occasions and people who would love this]

💼 USE CASES:
[How they can use this product]
```

---

### Section 4: Care & Shipping
```
🧹 CARE INSTRUCTIONS:
[How to care for the product]

🚚 SHIPPING:
[Shipping times and methods]

📦 PACKAGING:
[How it will arrive]
```

---

### Section 5: Call to Action
```
Thank you for supporting our small business! 

Questions? Send me a message - I typically respond within 24 hours.

Want more? Follow my shop for new products and discounts!
```

---

## Description Formulas

### Formula 1: Emotional + Logical
```
1. Hook (emotional connection)
2. Story (why you created it)
3. Details (logical specs)
4. Benefits (what they gain)
5. Social proof (reviews mention)
6. Call to action
```

### Formula 2: Feature + Benefit
```
1. Feature: [What it is]
   Benefit: [What it does for them]
   
2. Feature: [What it is]
   Benefit: [What it does for them]
   
3. Feature: [What it is]
   Benefit: [What it does for them]
```
